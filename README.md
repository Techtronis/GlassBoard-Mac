# GlassBoard-Mac

A dashboard for Google Glass configurations (only APKs for now). Currently works for the Explorer Edition.

## How to use

Please install the android drivers from the official [website](https://developer.android.com/tools/releases/platform-tools).

Click on Select Drivers in GlassBoard and select the Platform-Tools file you have downloaded. The directory will save, you just need to click refresh to load it back up.

Make sure your Google Glass is connected and press refresh to see your APKs. If nothing shows up, that means the computer can't detect the glass.

# Install Firmware

To Install Firmware, all you need is the firmware folder which contains system.img, boot.img and recovery.img.

Press on Install Firmware and select the firmware folder.

THIS WILL RESET YOUR GOOGLE GLASS, YOU WILL LOSE YOUR DATA.
