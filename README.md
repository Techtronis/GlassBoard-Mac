# GlassBoard-Mac

A dashboard for Google Glass configurations (only APKs for now). Currently works for the Explorer Edition.

## How to use

Please install the android drivers from the official [website](https://developer.android.com/tools/releases/platform-tools).

Click on Select Drivers in GlassBoard and select the Platform-Tools file you have downloaded. You'll need to do this everytime you boot the app.

Make sure your Google Glass is connected and press refresh to see your APKs. If nothing shows up, that means the computer can't detect the glass.
